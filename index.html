While the game is running, you can configure events through the extension config page

The mod allows you to set up your own events if you wish. I will guide you through making your own events:

  1. After running the mod for the first time, navigate to the minecraft directory and enter into the config directory. There is a file called streamparticipation.json. Open this file with any text editor.

  2. If you are going to create events, delete all text from "players": { to "allowedEvents": {

  3. Each event is written under allowedEvents in the following format: "NAME": { "enabled": true, "cost": 50-10000 in increments of 50, "command": "COMMAND TO RUN" },

  4. For instance, if I want to make an event that clears the weather for 600 bits, I would write the following after allowedEvents: "weather/clear": { "enabled": true, "cost": 600, "command": "/weather clear" },
